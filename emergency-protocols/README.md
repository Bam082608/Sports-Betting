## âš ï¸ Trigger Conditions

### Automatic Activation Triggers

Emergency protocols **MUST** be activated when any of these conditions are met:

#### Trigger 1: Consecutive Losses
```
ACTIVATE IF: 3 or more consecutive losing bets

Severity Levels:
- 3 losses in a row â†’ YELLOW ALERT (Mild intervention)
- 5 losses in a row â†’ ORANGE ALERT (Moderate intervention)
- 7+ losses in a row â†’ RED ALERT (Severe intervention)
```

#### Trigger 2: Bankroll Drawdown
```
ACTIVATE IF: Bankroll down 20% or more from peak

Severity Levels:
- 20-25% drawdown â†’ YELLOW ALERT
- 25-35% drawdown â†’ ORANGE ALERT
- 35%+ drawdown â†’ RED ALERT (Full system pause)
```

#### Trigger 3: Emotional Compromise
```
ACTIVATE IF: Any of the following detected:
- Revenge betting urges (wanting to "get it back")
- Chasing losses with increased stakes
- Deviating from protocols without justification
- Feeling frustrated, angry, or desperate
- Skipping analysis to rush bets
```

#### Trigger 4: System Breakdown
```
ACTIVATE IF: Any of the following occurs:
- Multiple protocol violations in single session
- Betting without proper analysis
- Ignoring goalie confirmations or data verification
- Portfolio rules violated
- ADHD execution protocol abandoned
```

### Manual Activation

Emergency protocols can also be activated **voluntarily** if:
- You feel something is "off" even without meeting triggers
- Pattern of "close losses" suggesting analysis breakdown
- External life stress impacting betting focus
- Sensing early warning signs before triggers met

**It's better to activate early than wait for full crisis.**

**File:** `emergency-protocols/stop-the-bleeding-protocol.md`

### Core Principles from Protocol

**Principle 1: Pause is Power**
- Immediate pause is non-negotiable
- 24-48 hours minimum for emotional triggers
- No "light betting" or "testing the waters"

**Principle 2: Ego is the Enemy**
- Accept that something is wrong
- No rationalizing or making excuses
- Humble assessment required

**Principle 3: Protect the Bankroll**
- Bankroll preservation > ego preservation
- Better to pause too long than resume too soon
- Can't bet if bankroll is gone

**Principle 4: Process Over Results**
- Focus on decision quality, not outcomes
- If process was sound, variance is acceptable
- If process was flawed, results don't matter

**Principle 5: Systematic Recovery Only**
- No "hero bets" to recover losses
- Gradual stake increases only
- Must earn way back to full stakes

## ðŸ’¡ Crisis Prevention Strategies

### Early Warning System

**Watch for these signs BEFORE triggers:**
- Win rate dropping below 50% over 10 bets
- Feeling rushed or pressured during analysis
- Skipping protocol steps to save time
- Checking bankroll more than once per day
- Thinking about "making back" losses
- Less enjoyment in the process

**Preventive Actions:**
- Voluntary 24-hour pause
- Re-read COMPLETE-LESSONS-LEARNED.md
- Review last 5 bets in detail
- Reduce stakes by 25% proactively
- Take day off from betting

### Psychological Safeguards

**Built into System:**
- Mandatory protocol adherence (no shortcuts)
- Automated stake reduction (removes ego)
- Forced pause periods (prevents chasing)
- Lessons documentation (learn from mistakes)
- Success criteria (objective recovery metrics)

**Personal Additions:**
- Share struggles with trusted friend/mentor
- Journal emotional state before betting
- Set maximum daily time limit for betting
- Maintain life balance (work, family, hobbies)
- Celebrate process wins, not just monetary wins
