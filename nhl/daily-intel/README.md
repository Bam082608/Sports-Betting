# NHL Daily Intel

*This directory contains game-day research for NHL*

---

## How to Use

Create a new file for each game day using the naming convention and template below.

---

## File Naming Convention

`[YYYY-MM-DD]-slate.md`

Example: `2024-01-15-slate.md`
