## ğŸ¯ KEY LESSONS LEARNED

### **Major Insights**

**1. Never Combine Opposites**
- Over + Under on same ticket = guaranteed single point of failure
- Split across tickets = multiple win paths
- Evidence: Dallas game (+$1.64 swing)

**2. Same Props, Different Totals**
- Use same player props as base
- Vary only the total (Over vs Under)
- Increases win probability across tickets

**3. Team Floor Events Kill Everything**
- When team completely fails, all tickets die
- This is why we skip Tier 3 games
- Evidence: Anaheim 0-7 disaster

**4. 4-6 Tickets > 10-11 Tickets**
- Too many tickets = diluted focus
- 4-6 tickets = manageable, focused
- Less analysis paralysis
- Faster execution

**5. Rush Hour Protocol Saves Bankroll**
- Simple bets when time-constrained
- Avoid complex tickets when distracted
- Quality over quantity

**6. Multiple Win Paths = Bankroll Protection**
- Even if one narrative fails, others can win
- Reduces total-loss nights
- Smoother variance
