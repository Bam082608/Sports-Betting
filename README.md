### 1. Pre-Game Preparation (2-3 hours before)
1. Review [APEXVIPER_GAME_DAY_TIMELINE.md](/protocols/APEXVIPER_GAME_DAY_TIMELINE.md)

3. Use specialized protocols as needed:
   - [Goalie Intelligence](/protocols/APEXVIPER_GOALIE_INTELLIGENCE.md)
   - [Fatigue Matrix](/protocols/APEXVIPER_FATIGUE_MATRIX.md)
   - [Altitude Edge](/protocols/APEXVIPER_ALTITUDE_EDGE.md)
   - [Public Money](/protocols/APEXVIPER_PUBLIC_MONEY.md)

### 3. Execution
1. Follow [APEXVIPER_ADHD_EXECUTION_PROTOCOL_v3.2.0.md](/protocols/APEXVIPER_ADHD_EXECUTION_PROTOCOL_v3.2.0.md)
2. Build portfolio via [APEXVIPER_PORTFOLIO_CONSTRUCTION.md](/protocols/APEXVIPER_PORTFOLIO_CONSTRUCTION.md)
3. Log bets in [tracking/bet-log.md](/tracking/bet-log.md)
4. Save execution summary to [executions/](/executions/)

### 4. Post-Game Review
1. Update [tracking/pattern-log.md](/tracking/pattern-log.md)
2. Add insights to [tracking/lessons-learned.md](/tracking/lessons-learned.md)
3. Complete [weekly-roundup-template.md](/tracking/weekly-roundup-template.md) on Sundays
