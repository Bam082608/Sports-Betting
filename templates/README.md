## How to Use Templates

### Post-Game Analysis Template
1. **Copy template** to `/tracking/post-game/YYYY-MM-DD-GAME.md`
2. **Fill in all sections** immediately after games end
3. **Commit to GitHub** within 1 hour of completion
4. **Review weekly** for pattern identification

### Battle Plan Template
1. **Copy template** to `/executions/YYYY-MM-DD-battle-plan.md`
2. **Fill in during pre-game analysis** (2-3 hours before first game)
3. **Use as execution guide** during bet placement
4. **Archive after completion** with results filled in
5. **Reference for next slate** to improve workflow
