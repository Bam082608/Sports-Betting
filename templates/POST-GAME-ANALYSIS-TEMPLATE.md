## üéì USAGE GUIDELINES

### When to Complete

**Timing:**
- Immediately after all games complete (within 1 hour)
- While details are fresh
- Before looking at next slate

**Don't Wait:**
- Memories fade fast
- Details get fuzzy
- Lessons get lost

### How to Use Template

1. **Copy template** to new file: `/tracking/post-game/YYYY-MM-DD-SLATE.md`
2. **Fill systematically** (don't skip sections)
3. **Be honest** (especially emotional state, protocol violations)
4. **Focus on learning** (not just results)
5. **Commit to GitHub** (creates accountability)

### Key Sections (Don't Skip These)

**Must Complete:**
- Financial summary (bankroll tracking)
- Ticket type breakdown (performance by type)
- Lessons learned (actionable takeaways)
- Protocol adherence (did you follow process?)

**Optional (if time limited):**
- Detailed game-by-game review (can abbreviate)
- Player performance notes (capture highlights only)

### Integration with Other Systems

**Update These After Analysis:**
- `/tracking/bet-log.md` - Add today's bets
- `/tracking/lessons-learned.md` - Add key lessons
- `/tracking/pattern-log.md` - Add any patterns identified
- `/playbook/README.md` - Update ticket type performance table
