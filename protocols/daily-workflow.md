# Daily Betting Workflow

*APEXVIPER System - Daily Operations*

---

## Morning (Pre-Slate Analysis)

### 1. Check the Slate
- Review today's NHL games
- Identify which Tier 1 stars are playing
- Note any back-to-back situations

### 2. Tier 1 Star Identification
- **Gauthier (ANA)** - Playing?
- **Bedard (CHI)** - Playing?
- **Robertson (DAL)** - Playing?
- **Hintz (DAL)** - Playing?

### 3. Research DailyFaceoff
- Confirm line combinations
- Check power play units
- Verify goalie matchups
- Note any injury updates

---

## Afternoon (Game Assessment)

### 1. Defensive Matchup Analysis
- Identify opponent shot-blocking tendencies
- Check if playing vs CAR/BOS/NJ/VGK (avoid D-men props)
- Review recent opponent shot totals

### 2. D-Men Identification
- Select 7 D-men across games for cross-game parlays
- Prioritize PP1 players (Heiskanen, etc.)
- Check for favorable matchups

### 3. Build Prop Lists
- Tier 1 stars: 2P+3P SOG splits
- Role players: 2+ SOG combos
- D-men: Cross-game opportunities

---

## Pre-Game (Final Confirmation)

### 1. Final Confirmations
- Verify lines haven't changed
- Confirm all players active
- Check late scratches on Twitter/X

### 2. Build Tickets
- **Ticket 1:** Tier 1 star 2P SOG props
- **Ticket 2:** Tier 1 star 3P SOG props
- **Ticket 3:** Role player combos
- **Ticket 4:** D-men cross-game parlays
- **Ticket 5:** Hedges (Unders if needed)

### 3. Place Bets
- Use correct stake sizing (A/B/C/D tiers)
- Double-check all selections
- Screenshot all tickets

---

## Post-Game (Results & Analysis)

### 1. Record Results
- Log all bets in tracking/bet-log.md
- Calculate net profit/loss
- Update bankroll in tracking/bankroll.md

### 2. Analyze Performance
- Which props hit? Which missed?
- Any patterns in failures?
- Update team DNA files with new intel

### 3. Document Lessons
- Add insights to lessons-learned.md
- Update star ratings if needed
- Note any hedge opportunities missed

---

## Monthly Operations

### 1. Export Transactions
- Download full betting history
- Categorize by sport/bet type
- Calculate monthly ROI

### 2. Tax Considerations
- If bankroll exceeds **$5,000**, consider withdrawals
- Document net profits for tax purposes
- Maintain withdrawal records

### 3. System Review
- Evaluate star performance
- Adjust tier classifications
- Update scaling milestones

---

## Quick Reference Checklist

- [ ] Morning: Check slate, identify stars, research DailyFaceoff
- [ ] Afternoon: Assess games, identify D-men, check matchups
- [ ] Pre-game: Final confirmations, build tickets, place bets
- [ ] Post-game: Record results, analyze performance
- [ ] Monthly: Export transactions, tax if over $5k
