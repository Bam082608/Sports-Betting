## OVERVIEW

When faced with 10+ games, analysis paralysis is the enemy. This system provides:
1. Three-tier filtering to prioritize games
2. Time budgets per tier
3. Decision tree for rapid filtering
4. Clear skip criteria

**Core Principle:** You don't need to analyze every game. Focus on games with clear edges.

---

## 1. THREE-TIER FILTERING STRUCTURE

### Tier Overview

| Tier | Games | Time Per Game | Total Time | Action |
|------|-------|---------------|------------|--------|
| **TIER 1: Deep Analysis** | 2-3 games | 15-20 min each | 30-60 min | Full protocol |
| **TIER 2: Quick Analysis** | 2-3 games | 5-10 min each | 10-30 min | Quick hits only |
| **TIER 3: Skip Entirely** | Remaining | 0 min | 0 min | No analysis |

### Time Budget Rules (ADHD-Friendly)

| Total Games on Slate | Tier 1 Max | Tier 2 Max | Total Analysis Time |
|---------------------|------------|------------|---------------------|
| 10-12 games | 2 games | 2 games | 60 min max |
| 13-15 games | 3 games | 3 games | 90 min max |
| 16+ games | 3 games | 3 games | 90 min max (hard cap) |

**HARD RULE:** Never exceed 90 minutes total analysis time regardless of slate size.

## 8. ADHD-FRIENDLY TIPS

### Avoiding Analysis Paralysis

1. **Use the worksheet** - Don't try to hold it in your head
2. **Set timers** - 15 min alarm for Tier 1, 5 min for Tier 2
3. **Hard stops** - When timer rings, STOP and move on
4. **Embrace skipping** - Most games should be Tier 3
5. **No FOMO** - Games you skip aren't "missed opportunities"

### Time Management Rules

| Rule | Implementation |
|------|----------------|
| **Never exceed 90 min total** | Set a 90-min master timer |
