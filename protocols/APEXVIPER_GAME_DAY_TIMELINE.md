## ADHD-FRIENDLY TIPS

### Task Chunking Rules
1. **Hard time limits** - Use timers, stop when they ring
2. **One task at a time** - Don't jump between phases
3. **Mandatory breaks** - Non-negotiable disconnects
4. **Decision gates** - Clear yes/no checkpoints
5. **Stop points** - Know when to stop analyzing

### Avoiding Analysis Paralysis
- Max 4 games analyzed per night
- Max 15 minutes per game
- If unsure, move on (don't force bets)
- "Good enough" analysis beats perfect paralysis

### Timer Suggestions
Set phone/watch alarms for:
- 10:30 AM (end slate overview)
- 11:15 AM (end goalie intel)
- 12:00 PM (mandatory break starts)
- 2:00 PM (Phase 2 starts)
- 4:00 PM (mandatory break starts)
- 5:00 PM (Phase 3 starts)
- 6:30 PM (bet placement deadline)
