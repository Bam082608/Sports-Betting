
**Types of Lessons:**

5. **Unlucky Losses:** Correct analysis but lost (variance)

**Use When:**
- After any surprising result (win or loss)

- Weekly roundup compilation

## ğŸ”„ Weekly Roundup Process

### Step-by-Step Sunday Workflow

**Time Required:** 30-45 minutes  
**Frequency:** Every Sunday, end of week

**Step 1: Gather Data (5 minutes)**
```
1. Open bet-log.md
2. Open bankroll.md
3. Open lessons-learned.md
4. Open pattern-log.md
5. Have calculator ready
```

**Step 2: Fill Template Metrics (10 minutes)**
```
1. Open fresh copy of weekly-roundup-template.md
2. Count total bets from bet-log.md
3. Calculate wins/losses/pushes
4. Calculate net P/L from bankroll.md
5. Calculate ROI (P/L / Total Risk)
6. Break down by category (player props, totals, etc.)
```

**Step 3: Analyze Best/Worst Bets (10 minutes)**
```
1. Identify top 3 winners
2. Document why each worked (protocol used, edge found)
3. Identify top 3 losers  
4. Document why each failed (lesson learned)
5. Look for commonalities
```

**Step 4: Pattern Review (10 minutes)**
```
1. Review pattern-log.md
2. Note any new patterns from this week
3. Update sample sizes for tracking patterns
4. Mark patterns as validated/rejected if threshold met
5. Plan integration of validated patterns
```

**Step 5: Forward-Looking Actions (5 minutes)**
```
1. List top 3 lessons for next week
2. Identify focus areas for improvement
3. Note any protocol changes needed
4. Set specific goals for next week
```

**Step 6: Save and Archive (5 minutes)**
```
1. Save completed weekly roundup
2. Name: YYYY-MM-DD-weekly-roundup.md
3. Move to executions/ or archive/YYYY-MM/
4. Update COMPLETE-LESSONS-LEARNED.md if major insights
5. Reset lessons-learned.md for new week (keep critical items)
```
