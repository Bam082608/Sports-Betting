## TREND FORMULAS QUICK REFERENCE

### Average SOG Calculation
```
L5 SOG/GP = Total SOG in Last 5 Games / 5
L10 SOG/GP = Total SOG in Last 10 Games / 10
```

### Delta Calculation
```
Delta = L5 Average - L10 Average

Interpretation:
- Delta > +1.0 = On Fire üî•
- Delta > +0.5 = Trending Up
- Delta between -0.5 and +0.5 = Stable
- Delta < -0.5 = Trending Down
- Delta < -1.0 = Cold Streak ‚ùÑÔ∏è
```

### Hit Rate Calculation
```
Hit Rate (2+ SOG) = Games with 2+ SOG / Total Games √ó 100%

Example: Player had 2+ SOG in 4 of last 5 games
Hit Rate = 4/5 √ó 100% = 80%
```

### Win Rate Calculation
```
Win Rate = Bets Won / Total Bets √ó 100%
```

### ROI Calculation
```
ROI = Net Profit / Total Wagered √ó 100%
```
