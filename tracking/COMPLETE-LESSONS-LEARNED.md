# ‚öñÔ∏è CRITICAL BETTING RULES - HARD LAWS

## Rule 1: The Titanic Rule (No Anchors)
**Effective Date:** November 23, 2025  
**Cause:** $34 weekend losses from anchor structure  
**Status:** PERMANENT LAW

```
DEFINITION: An "anchor" is any team that appears in more than one ticket.

LAW: Never put the same team in multiple tickets. EVER.

WHY: If the anchor fails, all tickets fail.
This converts diversification into concentration.
```

## Rule 2: The 3-Ticket Maximum
- Max 3 tickets per night/slate
- Focus on highest-edge opportunities
- Quality over quantity
- Easy to track and manage

**Exception:** Championship events (UFC PPV, March Madness) - still cap at 5 tickets maximum

## Rule 3: The Viper Gate
No bets placed without APEXVIPER verification.

## Rule 4: The Independence Test
Ask: "If Ticket 1 loses, can Tickets 2 and 3 still win?"
If NO ‚Üí Structure is anchored ‚Üí REBUILD

## Rule 5: The Banker Ban
Never treat any team as a "guaranteed win."
- -125 favorite = 55.6% implied probability
- That means they lose 44.4% of the time
- There are NO locks in hockey (or any sport)

## Rule 6: The Chalk Tax (Never Bet Favorites Worse Than -250)
- Heavy favorites (-300, -500, -750) have terrible risk/reward
- One upset/variance event = catastrophic loss
- Public loves favorites = lines get inflated

## Rule 7: Unit Sizing Limits
- Standard edge: 1-1.5u
- Strong edge: 2u
- Nuclear edge: 3u MAX
- Never risk >3u on any single bet (even "locks")

# üï∏Ô∏è PARLAY CONSTRUCTION STRATEGY

## The Core Problem We Solved
Random combination of picks creates anchors without realizing it.

## Step 1: Rank Your Picks (Confidence Tiers)

**A-TIER (70%+ confidence):**
- Your best picks
- Use as foundation, but ONCE each
- Example: B2B fade, backup goalie fade

**B-TIER (55-65% confidence):**
- Good picks, but not locks
- Example: Home favorite with rest

**C-TIER (45-55% confidence - Value Plays):**
- Underdogs with edge, lottery tickets
- Example: Home dog +200

## Step 2: Check Correlation

**AVOID CORRELATION:**

| Bad Parlay | Why Bad |
|------------|---------|
| Buffalo + Toronto | Atlantic Division rivals |
| Colorado + Vegas | Pacific Division, same night |
| Edmonton + Calgary | Alberta rivals |
| 3 B2B teams | Same schedule situation |

**PREFER INDEPENDENCE:**

| Good Parlay | Why Good |
|-------------|----------|
| Buffalo (Eastern) + Colorado (Western) | Different conferences |
| Toronto + Vegas | Different opponents, independent |
| Rested team + B2B team | Different schedule situations |

## Step 3: Build Layers

**2-leg parlays:** Use A+A or A+B (independent games)
**3-leg parlays:** Mix tiers (A+B+C, spread across conferences)
**4+ leg lottery:** Include lower tiers, expect to lose

## Step 4: Run Anchor Check
Count how many tickets each team appears in:
- No team in >50% of tickets
- Ideally each team in max 2-3 tickets

## Example Structure (5 Picks)

**Your picks tiered:**
```
A-Tier: Carolina (best pick)
A-Tier: Boston (second best)  
B-Tier: Seattle
B-Tier: Colorado
C-Tier: Vegas (value play)
```

**Build independent web:**
```
Ticket 1: Carolina + Boston (A+A) - $5
Ticket 2: Seattle + Colorado (B+B) - $3
Ticket 3: Carolina + Seattle + Vegas (A+B+C) - $2
Ticket 4: Boston + Colorado + Vegas (A+B+C) - $2
Ticket 5: All 5 lottery - $1
```

**Distribution check:**
- Carolina: Tickets 1, 3, 5 (3 tickets)
- Boston: Tickets 1, 4, 5 (3 tickets)
- Seattle: Tickets 2, 3, 5 (3 tickets)
- Each team in 3 tickets max = NO ANCHORS

# üí∞ BANKROLL MANAGEMENT

## Unit Sizing

```
Bankroll: $500
1u = $5 (1%)

Standard bet (1.5u): $7.50
High confidence (2u): $10
Max bet (3u): $15

After win streak ‚Üí Bankroll: $650
1u = $6.50 (recalculated)
Bets automatically scale up

After losses ‚Üí Bankroll: $400
1u = $4 (recalculated)
Bets automatically scale down (survival mode)
```

## Nightly Exposure Limits
- **Max exposure per night:** 6.5% of bankroll
- **Max per slate:** 2.5% of bankroll
- **Max per bet:** 3u (3% of bankroll)

## The 20-Unit Buffer
Always keep 20 maximum bet sizes in reserve:
- If betting $10/unit, keep $200+ in bankroll
- If betting $25/unit, keep $500+ in bankroll
- Survives worst-case losing streaks

## Stop-Loss Protocols
- **2 consecutive losses:** STOP immediately, review process
- **5+ unit loss in session:** Full system audit required
- **Bankroll drops 50%:** Mandatory break, reassess strategy
- **10-bet losing streak:** Mandatory cool-down

## Profit-Taking
- Withdraw 25-50% of profits regularly
- Prevents "playing with house money" mentality
- Secures gains for long-term wealth building
