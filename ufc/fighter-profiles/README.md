# UFC Fighter Profiles

*This directory contains betting profiles for UFC fighters*

---

## How to Use

Create a new file for each fighter you want to track using the template below.

---

## File Naming Convention

`[firstname-lastname].md`

Examples:
- `jon-jones.md`
- `islam-makhachev.md`
- `alex-pereira.md`
