# UFC Event Research

*This directory contains research for upcoming UFC events*

---

## How to Use

Create a new file for each event using the naming convention and template below.

---

## File Naming Convention

`[date]-[event-name].md`

Examples:
- `2024-01-20-ufc-fight-night-vegas.md`
- `2024-02-03-ufc-298.md`
